Contoh Penggunaan Karakter Khusus Pada HTML


  • &lt; (Kurang Dari) : Akan menampilkan karakter seperti ini [ < ]

  • &gt; (Lebih Dari) : Akan menampilkan karakter seperti ini [ > ]

  • &amp; (Ampersand) : Akan menampilkan karakter seperti ini [ & ]

  • &quot; (Tanda Kutip) : Akan menampilkan karakter seperti ini [ " ]

  • &apos; (Tanda Kutip Tunggal) : Akan menampilkan karakter seperti ini [ ' ]

  • &copy; (Copyright) : Akan menampilkan karakter seperti ini [ © ]

  • &trade; (Trademark) : Akan menampilkan karakter seperti ini [ ™ ]
