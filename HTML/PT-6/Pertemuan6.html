Cara Membuat Kopi Arabika Java Ijen Raung :

  1. Masak air hingga mendidih.
  2. Tambahkan bubuk kopi Arabika Java Ijen Raung ke dalam air mendidih.
  3. Rebus selama 3-4 menit dengan api kecil, agar aroma kopi keluar sempurna.
  4. Matikan api dan diamkan selama 1 menit agar ampas mengendap.
  5. Tuang kopi ke dalam cangkir menggunakan saringan untuk memisahkan ampasnya.
  6. Tambahkan gula atau susu sesuai selera, lalu aduk hingga rata.
  7. Nikmati kopi selagi hangat!
